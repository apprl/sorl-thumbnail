{% trans "Festivalsommar med JC" %}

{% trans 'Tävlingen öppen 6 juni - 30 augusti.' %}

{% blocktrans %} Under hela sommaren kan du tävla med JC och Nöjesguiden om en hel festivaloutfit ur JC:s senaste festivalkollektion. {% endblocktrans %}

{% blocktrans %} JC har tagit fram en festivalkollektion till sommaren 2014 och kommer tillsammans med Nöjesguiden att besöka landets alla största festivaler. Nu har du chans att vinna kläder från JC till ett värde av 3 000:-. {% endblocktrans %}

{% blocktrans %} För att vara med och tävla vill vi att du skapar en look med minst ett av plaggen i kollektionen som bas. De tio looks som har flest ❤ går sedan till final, där JC och Nöjesguiden väljer ut tre vinnare som får varsitt presentkort på 3000 kronor. {% endblocktrans %}

  1. {% url 'shop' as shop_url %} {% url 'look-create' as look_url %} {% url 'auth_register' as register_url %} {% url 'contest-jc-charts' as jc_toplist_url %} {% blocktrans %} Skapa ett konto på APPRL och skapa en collage look här. {% endblocktrans %}

  3. {% blocktrans %} Publicera din look på Apprl och dela den med dina vänner för att få fler ❤! {% endblocktrans %}
