APPRL API

Parameters

sid
To earn money on your links you need an SID, which you get by becoming an APPRL Publisher. Apply here: http://apprl.com/publisher/. OPTIONAL but RECOMMENDED
page
Between 1 and last page number. OPTIONAL
limit
Between 1 and 30, default is 10. OPTIONAL
category_name
One category name in text, case insensitive (show category names). OPTIONAL
color_name
One or many color names in text, seperated with comma, case insensitive (show color names). OPTIONAL
brand
One brand name in text, case insensitive (see all our brands here). OPTIONAL
gender
Either 'M' for man or 'W' for woman. OPTIONAL

Use /sv/ to return data in Swedish and SEK currency
Use /en/ to return data in English and EUR currency

Example: http://api.apprl.com/en/products/?sid=250&limit=30&category_name=shoes&gender=M
