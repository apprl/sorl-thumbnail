APPRL API

Parameters

sid
You need an SID to earn money on your links, which you get by becoming an APPRL Publisher. Apply here if you're interested: http://apprl.com/publisher/. OPTIONAL but RECOMMENDED
page
Between 1 and last page number. OPTIONAL
limit
Between 1 and 30, default is 10. OPTIONAL
category_name
English category name in text (show category names). OPTIONAL
color_name
English color name in text (show color names). OPTIONAL
brand
Brand name in text. OPTIONAL
gender
Either 'M' for man or 'W' for woman. OPTIONAL

Example: http://api.apprl.com/en/products/?sid=250&limit=30&category_name=shoes&gender=M
